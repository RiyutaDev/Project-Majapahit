#Project Majapahit – Sistem Informasi Apotek

Sistem Informasi Apotek berbasis web yang dikembangkan menggunakan CodeIgniter 4 untuk mendukung manajemen operasional apotek secara terintegrasi, mulai dari pengelolaan data obat, pemesanan, upload resep, laporan keuangan, hingga sistem approval pembelian.

#Teknologi yang Digunakan

Backend: CodeIgniter 4

Frontend: Bootstrap 5 + TailwindCSS

Database: MySQL / MariaDB

Server: Apache / Nginx

PHP Version: >= 8.1

Version Control: GitHub
